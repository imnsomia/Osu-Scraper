# Osu-Scraper

A simple API to query user data in osu without authentication or rate limit via the osu website

## Prerequisites
* BeautifulSoup

command to install it
`pip3 install beautifulsoup4`

## Installing Osu-Scraper

clone the repo and use it as you like
* git clone https://github.com/imnsomia/Osu-Scraper/
